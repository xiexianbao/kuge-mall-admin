# 酷鸽商城后台管理前端

## 开发工具
* node-20.8.0
* pnpm-8.8.0
* 淘宝镜像源：https://registry.npmmirror.com

## 启动命令
```shell
# 安装依赖
pnpm install

# 本地运行
pnpm run dev
```
